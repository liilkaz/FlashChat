# ⚡️FlashChat

* Проект на Swift 5
* Минимальная поддерживаемая iOS – 13
* Только iPhone
* Портретная ориентация
* Верстка UIKit программно
* Firebase

**Проект выполнен в рамках марафона "SWIFT MARATHON 6.0"** -> https://t.me/swiftmarathon
